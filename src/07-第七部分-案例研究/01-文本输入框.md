# 第 19 章：手写一个文本输入框

本章基于 `examples/input.rs`（747 行）和 `src/input.rs`（`InputHandler` trait），讲解如何从零实现一个功能完整的文本输入框。

> 源文件参考：`examples/input.rs`、`src/input.rs`

## InputHandler trait 完整定义

```rust
// src/input.rs:12-72
pub trait InputHandler: 'static {
    fn text_for_range(
        &mut self,
        range: Range<usize>,
        adjusted_range: &mut Option<Range<usize>>,
        window: &mut Window,
        cx: &mut App,
    ) -> Option<String>;

    fn selected_text_range(
        &mut self,
        ignore_disabled_input: bool,
        window: &mut Window,
        cx: &mut App,
    ) -> Option<UTF16Selection>;

    fn marked_text_range(
        &self,          // &self，不是 &mut self
        window: &mut Window,
        cx: &mut App,
    ) -> Option<Range<usize>>;

    fn replace_text_in_range(
        &mut self,
        replacement_range: Option<Range<usize>>,
        text: &str,     // &str，不是 SharedString
        window: &mut Window,
        cx: &mut App,
    );

    fn replace_and_mark_text_in_range(
        &mut self,
        replacement_range: Option<Range<usize>>,
        new_text: &str, // &str，不是 SharedString
        new_selected_range: Option<Range<usize>>,
        window: &mut Window,
        cx: &mut App,
    );

    fn unmark_text(
        &mut self,
        window: &mut Window,  // 有 window 参数
        cx: &mut App,
    );

    fn bounds_for_range(
        &mut self,
        range: Range<usize>,
        window: &mut Window,
        cx: &mut App,
    ) -> Option<Bounds<Pixels>>;

    fn character_index_for_point(
        &mut self,
        point: Point<Pixels>,
        window: &mut Window,
        cx: &mut App,
    ) -> Option<usize>;
}
```

关键点与常见误解：
- **每个方法都有 `window: &mut Window` 参数**（除了 `marked_text_range` 的 `&self` 例外）。
- 文本参数是 `&str`（不是 `SharedString`）。
- `selected_text_range` 第一个参数是 `ignore_disabled_input: bool`。
- `text_for_range` 有额外的 `adjusted_range: &mut Option<Range<usize>>` 参数。
- 共 **8 个方法**（早期版本可能只有 6 个，0.2.2 增加了 `bounds_for_range` 和 `character_index_for_point`）。
- `marked_text_range` 使用 `&self`（不可变借用）而非 `&mut self`。

## 整体架构

```
struct InputElement {
    focus_handle: FocusHandle,
    editor: Entity<Editor>,
}

struct Editor {
    text: SharedString,
    selection: Option<Range<usize>>,
    marked_text: Option<(Range<usize>, SharedString)>,
}
```

两个实体分工：
- **`Editor`**：数据模型——文本内容、选区、marked text。实现 `InputHandler` trait。
- **`InputElement`**：UI 桥接——持有 `FocusHandle` 和 editor 引用，实现 `Element` 和 `Render` trait。

## Element trait 实现要点

`InputElement` 需要实现 `Element`（三阶段）和 `Render`：

```rust
impl Render for InputElement {
    fn render(&mut self, window: &mut Window, cx: &mut Context<Self>) -> impl IntoElement {
        div()
            .track_focus(&self.focus_handle)
            .key_context("Input")
            // 注册所有按键动作的 listener
            .on_action(cx.listener(Self::handle_backspace))
            .on_action(cx.listener(Self::handle_delete))
            .on_action(cx.listener(Self::handle_move_left))
            // ...
    }
}
```

`cx.listener()` 接受一个方法引用，该类方法签名为 `Fn(&mut Self, &ActionType, &mut Window, &mut Context<Self>)`。

## IME 输入的关键点

IME 组合输入通过 `InputHandler` trait 的以下方法协作：

1. **`marked_text_range()`**：返回当前组合输入的文本范围（用于渲染下划线）。
2. **`replace_text_in_range()`**：IME 确认文本时的回调——替换指定范围或追加新文本。
3. **`replace_and_mark_text_in_range()`**：组合输入中——标记未确认的文本。
4. **`unmark_text()`**：组合输入结束——清除标记。

在元素的 `paint()` 阶段，通过 `window.handle_input(&focus_handle, input_handler, cx)` 注册输入处理器（`src/window.rs:3927`）。

## 剪贴板操作

```rust
// 复制
fn copy_selection(&self, cx: &mut Context<Self>) {
    if let Some(selection) = &self.selection {
        let text = self.text[selection.clone()].to_string();
        cx.write_to_clipboard(ClipboardItem::new_string(text));
    }
}

// 粘贴
fn paste(&mut self, cx: &mut Context<Self>) {
    if let Some(item) = cx.read_from_clipboard() {
        if let Some(text) = item.text() {
            self.insert_text(&text, cx);
        }
    }
}
```

## 关键收获

1. **`InputHandler` 有 8 个方法，每个都带 `window: &mut Window` 参数**。
2. **文本参数是 `&str`，不是 `SharedString`**。
3. **`marked_text_range` 使用 `&self`（只读），其他方法使用 `&mut self`**。
4. **通过 `window.handle_input()` 注册，不是 `EntityInputHandler`**（0.2.2 中 trait 名是 `InputHandler`）。
