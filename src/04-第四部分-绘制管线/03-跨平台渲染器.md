# 第 13 章：跨平台渲染器对比

GPUI 支持三个 GPU 渲染后端：macOS Metal、Windows DirectX 11、和跨平台 Blade。

> 源文件参考：`src/platform/mac/metal_renderer.rs`、`src/platform/windows/directx_renderer.rs`、`src/platform/blade/blade_renderer.rs`、`build.rs`

## 渲染器消费统一输入

```
Window::draw()
  └── Scene::finish() → Scene::batches()
        │
        ├── macOS: render(&mut metal_renderer)
        ├── Windows: render(&mut directx_renderer)
        └── Linux:   render(&mut blade_renderer)

统一输入：PrimitiveBatch 枚举变体（切片引用）
统一输出：framebuffer（呈现到平台窗口）
```

## Shader 类型对比

不管哪个平台，这些 shader 做同样的事：

| Shader 类型 | 图元 | 主要逻辑 |
|------------|------|---------|
| **Quad** | `PrimitiveBatch::Quads` | 背景填充、边框、圆角裁剪 |
| **Shadow** | `PrimitiveBatch::Shadows` | 高斯模糊盒阴影 |
| **Path** | `PrimitiveBatch::Paths` | 从 lyon tessellation 结果填充路径 |
| **MonochromeSprite** | 单色精灵批次 | 采样 atlas 纹理 × 颜色 |
| **PolychromeSprite** | 彩色精灵批次 | 采样 atlas 纹理（保留原色） |
| **Underline** | `PrimitiveBatch::Underlines` | 直线或波浪线填充 |
| **Surface** | `PrimitiveBatch::Surfaces` | 原生系统表面组合 |

| | Metal (macOS) | DX11 (Windows) | Blade (跨平台) |
|---|---|---|---|
| **Shader 语言** | MSL | HLSL | WGSL |
| **编译时机** | 构建时 (xcrun → .metallib) | 构建时 (fxc.exe → .cso) | 构建时验证 (naga)；运行时编译 |
| **源文件** | `platform/mac/shaders.metal` | 内嵌 Rust 常量 | `platform/blade/shaders.wgsl` |

## Metal 渲染器

```rust
// src/platform/mac/metal_renderer.rs:99-120（简化）
pub(crate) struct MetalRenderer {
    layer: metal::MetalLayer,
    presents_with_transaction: bool,
    // 内部结构复杂的字段…
    sprite_atlas: Arc<MetalAtlas>,
    instance_buffer_pool: InstanceBufferPool,
    unit_vertices: metal::Buffer,
    // 八个独立的 pipeline state（不是 HashMap）
    path_rasterization_pipeline_state: metal::RenderPipelineState,
    path_sprites_pipeline_state: metal::RenderPipelineState,
    shadows_pipeline_state: metal::RenderPipelineState,
    quads_pipeline_state: metal::RenderPipelineState,
    underlines_pipeline_state: metal::RenderPipelineState,
    monochrome_sprites_pipeline_state: metal::RenderPipelineState,
    polychrome_sprites_pipeline_state: metal::RenderPipelineState,
    surfaces_pipeline_state: metal::RenderPipelineState,
    // 路径渲染中间纹理
    path_intermediate_texture: metal::Texture,
    path_intermediate_msaa_texture: metal::Texture,
    path_sample_count: u32,
    // ...
}
```

关键点：
- **管线不是 HashMap**：八种图元类型各有独立的具名 `RenderPipelineState` 字段。
- **atlas 是 `Arc<MetalAtlas>`**：不是 `Vec<metal::Texture>`。
- **`layer: metal::MetalLayer`** 是渲染目标（`CAMetalLayer`）。

## DirectX 11 渲染器

```rust
// src/platform/windows/directx_renderer.rs:39-48（简化）
pub(crate) struct DirectXRenderer {
    hwnd: HWND,
    devices: ManuallyDrop<DirectXRendererDevices>,   // 包装了 device, context 等
    resources: ManuallyDrop<DirectXResources>,        // 包装了 swap_chain 等
    atlas: Arc<DirectXAtlas>,
    globals: DirectXGlobalElements,
    pipelines: DirectXRenderPipelines,
    direct_composition: Option<DirectComposition>,
    font_info: &'static FontInfo,
}
```

关键点：
- **设备被组合到 `DirectXRendererDevices`** 中：包含 `adapter`、`dxgi_factory`、`device`、`device_context`、`dxgi_device`。
- **资源被组合到 `DirectXResources`** 中：包含 `swap_chain` 和其他 swap chain 资源。
- **`atlas: Arc<DirectXAtlas>`**：图集通过 `Arc` 共享。

## Blade 渲染器

```rust
// src/platform/blade/blade_renderer.rs:327-344（简化）
pub struct BladeRenderer {
    gpu: Arc<gpu::Context>,
    surface: Option<gpu::Surface>,
    surface_config: gpu::SurfaceConfig,

    last_sync_point: gpu::SyncPoint,
    pipelines: BladeRenderPipelines,
    instance_belt: gpu::InstanceBelt,
    atlas: Arc<MetalAtlas>,  // 或 DirectX Atlas，取决于平台
    atlas_sampler: gpu::Sampler,
    // 路径渲染中间纹理
    path_intermediate_texture: gpu::Texture,
    path_intermediate_texture_view: gpu::TextureView,
    // macOS 上的 Core Video 纹理缓存

    core_video_texture_cache: Option<core_video::CVImageBufferTextureCache>,
    // ...
}
```

`BladeRenderer` 是**唯一 `pub` 的结构体**（其他都是 `pub(crate)`）。

## build.rs：Shader 编译

### macOS Metal

```
1. cbindgen 从 Rust 源码生成 C header (scene.h) —— 确保 Rust 和 MSL 的结构体内存布局一致
2. shaders.metal 中 #include "scene.h"
3. xcrun -sdk macosx metal → .air
4. xcrun -sdk macosx metallib → .metallib
```

### Windows DirectX

```
# 每种 shader 独立的编译命令
fxc.exe /T vs_5_0 /E vs_main quad.hlsl /Fo quad_vs.cso
fxc.exe /T ps_5_0 /E ps_main quad.hlsl /Fo quad_ps.cso
# 结果嵌入 Rust 字节数组
```

### Blade

```
build.rs 用 naga::front::wgsl::parse_str 验证 shaders.wgsl
运行时由 blade-graphics 编译为目标平台原生 shader
```

## Cargo features

```toml
[features]
macos-blade = []       # macOS 上用 Blade 代替 Metal
runtime_shaders = []   # Metal shader 运行时编译（开发用）
```

- **默认**：macOS=Metal，Windows=DX11，Linux=Blade
- **`macos-blade`**：macOS 上切换到 Blade
- **`runtime_shaders`**：允许修改 shader 后无需重新编译

---

**如果你只记住一件事：** 三个渲染器都消费 `Scene::batches()` 返回的 `PrimitiveBatch` 枚举变体。Metal renderer 有八个独立的 pipeline state 字段（不是 HashMap）。DirectX renderer 将设备/资源组合到包装结构体中。只有 `BladeRenderer` 是 `pub` 的。
