# 附录 A：关键类型速查

## 核心应用类型

```
Application          — 应用程序入口。创建 → 配置 → run()
App                  — 全局状态管理器（EntityMap、Windows、Globals、Keymap）
AppCell(RefCell<App>)— App 的运行时可变容器
```

## 上下文（Context）层次

```
Context<'a, T>       — 实体级借用上下文 (&'a mut App + WeakEntity<T>)
  ├─ notify()        — 标记实体已变更
  ├─ emit(E)         — 发射类型化事件
  ├─ new(Fn)         — 创建新实体
  ├─ spawn(Fn)       — 启动异步任务
  ├─ observe()       — 观察 notify
  ├─ subscribe()     — 订阅事件
  ├─ to_async()      — → AsyncApp
  └─ focus_handle()  — → FocusHandle

AsyncApp             — 'static + Send 的 async 上下文
  ├─ update(Fn)      — 回到主线程（→ Result<_, _>）
  ├─ read_with(Fn)   — 只读回到主线程
  └─ spawn(Fn)       — 再次 spawn 异步任务

AsyncWindowContext   — AsyncApp + AnyWindowHandle
  └─ update(Fn)      — 回到主线程 + &mut Window

TestAppContext       — 测试专用（自有 AppCell, 模拟器）
  ├─ run_until_parked() — 等待所有异步完成
  └─ dispatch_keystroke() — 模拟按键
```

## 实体系统

```
Entity<T>            — 强类型句柄 (EntityId + Arc<Weak<...>>)
WeakEntity<T>        — 弱引用（不阻止释放）
AnyEntity            — 类型擦除句柄
EntityMap            — slot map 存储所有实体
EntityId             — slotmap key

Reservation<T>       — 预留槽位（先占位后填入）
GpuiBorrow<'a, T>    — RAII 租约 guard
```

## Element 核心

```
Element              — 三阶段 trait
  ├─ request_layout() → (LayoutId, State)
  ├─ prepaint(bounds) → State
  └─ paint(bounds)

IntoElement          — Into-like 转换
Render               — View trait (fn render → impl IntoElement)
RenderOnce           — 一次性组件 trait
ParentElement        — 容器 trait (child/children)
InteractiveElement   — 交互 trait (on_click/on_drag/...)
StatefulInteractiveElement — 有状态的交互
Styled               — 样式链式 API

Drawable<E>          — Element 的类型状态机封装
AnyElement           — 类型擦除元素
GlobalElementId      — SmallVec<[ElementId; 32]> (ID 路径)
```

## 内置元素

```
Div                 — 容器（flexbox/grid）
Label / Headline    — 文本
Img                 — 图片/动画
Canvas              — 自定义绘制
List                — 滚动列表
UniformList         — 虚拟化列表
Anchored            — 覆盖层/弹出层
Animation           — 关键帧动画
Deferred            — 延迟渲染
Svg                 — SVG 渲染
Surface             — 原生表面
Empty               — 零尺寸占位
```

## Window 系统

```
Window              — 平台窗口的 GPUI 封装
WindowHandle<V>     — 类型化窗口句柄（Copy）
AnyWindowHandle     — 类型擦除窗口句柄
WindowOptions       — 窗口创建配置
WindowBounds        — Windowed/Maximized/Fullscreen
WindowAppearance    — Light/Dark/Vibrant variants
FocusHandle         — 焦点句柄（Arc<FocusState>）
WeakFocusHandle     — 弱焦点句柄（Weak<FocusState>）
DispatchPhase       — Capture/Bubble
KeystrokeEvent      — 解析后的按键事件
```

## Scene 与绘制

```
Scene               — 绘制命令列表（非树！）
  ├─ paint_operations: Vec<PaintOperation>
  ├─ quads: Vec<Quad>
  ├─ shadows: Vec<Shadow>
  ├─ paths: Vec<Path<Pixels>>
  ├─ underlines: Vec<Underline>
  ├─ monochrome_sprites: Vec<MonochromeSprite>
  ├─ polychrome_sprites: Vec<PolychromeSprite>
  └─ surfaces: Vec<PaintSurface>

PrimitiveKind       — Shadow/Quad/Path/Underline/
                      MonochromeSprite/PolychromeSprite/Surface

DrawOrder           — { layer_depth, z_index, primitive_kind }

ContentMask<Pixels> — 矩形裁剪区域

TransformationMatrix— 2×2 旋转/缩放 + 平移
```

## 几何与单位

```
Point<T>            — { x, y }
Size<T>             — { width, height }
Bounds<T>           — { origin: Point<T>, size: Size<T> }
Edges<T>            — { top, right, bottom, left }
Corners<T>          — { top_left, top_right, bottom_right, bottom_left }
Inset<T>            — { top, right, bottom, left } 用于偏移
Axis                — Horizontal / Vertical

Pixels(f32)         — 屏幕像素
DevicePixels(i32)   — 设备像素（HiDPI 底层）
ScaledPixels(f32)   — 缩放后像素
Rems(f32)           — 相对 root em
Em(f32)             — 相对当前 em
AvailableSpace      — Definite/MinContent/MaxContent
Length              — Definite/Auto
DefiniteLength      — Absolute(AbsoluteLength)/Fraction(f32)
AbsoluteLength      — Pixels/Rems/Em
```

## 样式系统

```
Style               — 完整样式规格
StyleRefinement     — Option<T> 链式修改
TextStyle           — 文本样式
TextStyleRefinement — 文本样式修改

Display             — Block/Flex/Grid/None
FlexDirection       — Row/Column
FlexWrap            — NoWrap/Wrap
AlignItems          — Start/End/Center/Baseline/Stretch
JustifyContent      — Start/End/Center/SpaceBetween/SpaceAround
Overflow            — Visible/Hidden/Scroll

BorderStyle         — Solid/Dashed
BoxShadow           — { blur, spread, offset, color }
```

## 颜色

```
Rgba                — { r, g, b, a } (f32, 0-1)
Hsla                — { h, s, l, a }
Background          — Solid(Hsla) | LinearGradient(...) | PatternSlash(...)
ColorSpace          — Srgb / Oklab
Colors / GlobalColors — 语义颜色方案
```

## 平台抽象

```
Platform            — 核心平台 trait (~30 个方法)
PlatformWindow      — 原生窗口句柄
PlatformDisplay     — 显示器信息
PlatformDispatcher  — 线程调度
PlatformTextSystem  — 文本引擎
PlatformAtlas       — GPU 图集管理
PlatformRenderer    — 场景渲染器（非 trait，每个平台具体实现）

// macOS: MacPlatform, MacWindow, MacDisplay, MacDispatcher
// Windows: WindowsPlatform, WindowsWindow, WindowsDisplay, WindowsDispatcher
// Linux: WaylandClient, X11Client, HeadlessClient
```

## 动作系统

```
Action              — 动作 trait
ActionRegistry      — 全局注册表 (inventory)
Keymap              — 按键绑定
KeyBinding          — 单个绑定
KeyContext          — 上下文匹配表达式
NoAction            — 解绑标记
```

## 异步

```
BackgroundExecutor  — 线程池（Send 任务）
ForegroundExecutor  — 主线程（!Send 任务）
Task<T>             — 可取消 future
  ├─ await()        — 等待结果
  ├─ detach()       — fire-and-forget
  └─ cancel()       — Drop
Subscription        — 观察/订阅的 RAII guard
```

## 资产与文本

```
AssetSource         — 资产加载源 trait
Asset<T>            — 类型化异步加载
RenderImage         — 解码后的图像数据
ImageSource         — Uri/Path/Embedded

TextSystem          — 字体引擎
FontId / FontFamilyId — 字体标识
FontMetrics         — 字体度量
ShapedGlyph         — 已塑形字形
ShapedRun           — 已塑形运行段
LineLayout          — 行布局结果
TextRun             — 带样式的文本段
HighlightStyle      — 高亮样式
```

## 常用 derive 宏

```
#[derive(Render)]    — 生成 Render trait impl
#[derive(IntoElement)]— 生成 IntoElement impl
#[derive(Action)]    — 生成 Action trait impl + 注册
#[derive(Focusable)] — 生成 Focusable trait impl
```
